Please note that you are not protected yet! You must install the CENO Router extension and activate it before you proceed to browse the web.

Currently, the CENO Router browser extension can run in Firefox or Chrome/Chromium. Please click the button corresponding to the browser you are using to view this page.

CENO Router

The CENO Router is a browser extension that, when activated, makes sure that all of your browsing activity happens through CENO. This guarantees that no information about the pages you are looking at leaks to censors.

Browser profiles

To avoid interfering with your regular browsing activity, we have created a new browser profile for you to install CENO Router in. You'll notice that this browser window is in a default state. When you start your browser again normally, it will not be the CENO version running, but your usual browser.

Firefox instructions

Step 1

The first step is to open a new tab, enter about:addons into the address bar and press enter/return.

Step 2

Click the gear/settings icon and then click Install Add-on From File... and select ceno/builds/cenorouter@equalit.ie-X.Y.Z.xpi.

Step 3

In the popup dialog that appears, click install.

↓ Finishing up

Chrome instructions

Step 1

In a new tab, enter chrome://extensions into the address bar and press enter/return.

Step 2

Check the Developer mode checkbox.

Step 3

Click the now-available Load unpacked extension... button and then open the ceno/ceno-chrome directory in the file browser that appears.

↓ Finishing up

You're done!

You should now have an icon in your toolbar for the CENO Router extension. Clicking on it reveals a popup with some information and a Toggle CENO button that, when clicked, will activate CENO if it is inactive and deactivate it otherwise. You must have the CENO Client running to be able to activate CENO.
